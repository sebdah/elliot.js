Welcome to {elliot.js}

{elliot.js} was created to fill a need for graphing near real time events in various web projects. It has been used to produce graphs for log analytics, customer actions, monitoring and similar services.

Special needs? {elliot.js} is highly configurable. You can easily change background colors, bar sizes, colors, fonts, update frequenzy etc. Have a look at our examples for some inspiration.

You are free to use {elliot.js} in your open source projects, for educational or evaluations purposes. If you are using {elliot.js} for commercial purposes, please refer to the pricing section for more details.

{elliot.js} relies on the HTML 5 <canvas> and regular JavaScript. It is widely supported, both on mobile platforms and desktop browsers. Have a look at http://caniuse.com/#feat=canvas for more detailed information about <canvas> browser support.

Feature list

Excited? Me too! Here are some of {elliot.js}'s highlights:
  • Near real time graphing
  • Auto scaling for large data sets
  • Free (as in free beer) for open source projects
  • Configurable update frequency
  • Makes use of the powerful hardware supported HTML5 <canvas> tag
  • Support for all relevant browsers. Read more at http://caniuse.com/#feat=canvas
  • Highly configurable design options, including colors, sizes and fonts makes elliot a web cameleont

As you might have noticed. We initiate {elliot.js} by calling Elliot() which takes two arguments; canvas tag id and a configuration object. The next thing we need to do is to add data to the graph. This is done with a simple JavaScript call to elliot.add(float) or elliot.remove(float). In the configuration above, we set incrementalValues to true, which means that {elliot.js} will keep track of the current value all the time and all you need to do is to adjust it with the add() and remove() functions. Here's an example:

{% highlight javascript %} elliot.add(10); elliot.remove(5); {% endhighlight %} Boring example? Indeed!
